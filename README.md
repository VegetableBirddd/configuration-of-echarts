## 📣 简介

configuratio-of-echarts 是一个用于在线观看echarts图表的demo，创建的目的在于为开发者提供便利。如果对echarts不太熟悉，这将是一个很好的图形化页面来搭配自己想要的图表。使用者可以快速点击按钮为自己的图表进行属性搭配，也可以进行代码的输入。结果会输出一个配置项，供开发者CV操作。在线demo地址： ...

## 📝 使用

项目启动：npm start
项目打包：npm run build
